# 风格

* 中英文之间使用一个空格分割
* [参考文章](https://github.com/sparanoid/chinese-copywriting-guidelines)

# 术语

部分技术术语，我们无需进行翻译，应该保留。

* Jenkins
* Pull Request
* master
* token
