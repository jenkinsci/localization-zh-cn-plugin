扩展为不同用途添加新功能或修改现有插件行为。 扩展帮助用户更精确地调整插件行为以满足他们的需求。

扩展包括:

  • 克隆Clone 扩展 修改从远程获取更改并将其集成到代理工作区中的 Git 操作。 这些扩展可以调整所获取的历史记录的数量、允许获取操作运行的时间长度以及其他获取细节。
  • 检出Checkout 扩展 修改将文件从代理上的 Git 仓库放置到工作区中的 Git 操作。 这些扩展可以调整检出操作的最大持续时间、Git 子模块的使用和行为、工作区在磁盘上的位置等更多内容。
  • 变更历史Changelog 扩展 为不同情况调整源代码差异计算。
  • 标签Tagging 扩展 允许插件在当前工作区应用标签。
  • 构建启动 扩展 控制启动构建的条件。 他们在轮询时可以忽略变更通知，或者强制对提交进行更深入的评估。
  • 合并Merge 扩展 可选地将其他分支的更改合并到代理工作区的当前分支中。 他们控制用于合并的源分支以及应用于合并的选项。
