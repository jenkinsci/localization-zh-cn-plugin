参数的类型：
  • 单选：用户从下拉菜单中选择一个选项，该菜单由显式值（见下面的值字段）或属性文件（见下面的属性文件和属性键字段）填充。

  • 多选：用户可以从多行框中选择多个选项，该框由显式值（见下面的值字段）或属性文件（见下面的属性文件和属性键字段）填充。

  • 复选框：用户可以选择零个或多个复选框，这些复选框由显式值（见下面的值字段）或属性文件（见下面的属性文件和属性键字段）标记。

  • 多级单选：用户从下拉菜单中选择一个选项，然后会出现另一个下拉菜单，显示依赖于第一个值的选择项；根据第二个选择，可能会出现第三个下拉菜单，显示依赖于前两个选择的选项，依此类推。 属性文件是一个制表符分隔的文件，级别在列中定义，选择项在行中定义。例如，要进行两级选择（首先选择国家/地区，然后选择城市），可以指定如下所示的文件：

      国家	城市
  中国	杭州
  中国  上海
  中国	台湾
  日本	东京
    这将导致第一个下拉菜单的选项为“选择一个国家...”，“美国”，和“墨西哥”（初始选择是“选择一个国家...”，它作为下拉菜单的标签）。用户选择一个国家后，“城市”下拉菜单将会出现。如果首先选择了美国，那么旧金山和芝加哥将是可选项；但如果选择了墨西哥，那么墨西哥城和坎昆将是可选项。

    表示级别的列必须在值字段中指定。例如，“Country,City”（国家，城市）可以是有效的值。

    请注意，多级选择不支持默认值。

  • 多级多选：与单选相同，但在选择完所有级别后，会出现一个“再选择另一个...”的按钮，并且会展示额外的多级选择选项。
